# 速享

使用 Flutter 开发，实现局域网中多个设备间文件的快速共享，查看文件的设备不需要安装任何的软件，只需要有浏览器即可。
8001与8002端口的区别在 app 中有提示，可针对个人的情况使用对应的端口。

可快速创建文件共享窗口，安装此客户端的所有设备可通过窗口的url加入，也可直接通过浏览器打开。

## 功能列表

- 通过 web 浏览文件
- 图片在线浏览，视频在线播放
- 快速获得二维码
- 像聊天一样互传文件
- 扫码连接
- 物理设备下载网速显示
